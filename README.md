# planilha-de-plano-alimentar-com-controle-de-calorias
Criação de planilha de plano alimentar com controle de calorias em python feito pelo claude

# Comando Inicial: 
criar uma planilha de plano alimentar com controle de calorias ingeridas, uma tabela com alimentos distribuidos por categorias ( carboidratos, proteinas, saladas, frutas, bebidas, legumes. etc.. e outros que vc achar necessários para uma dieta saudavel, uma coluna que mostre o valor de cada grama dos alimentos e suas calorias, uma planilha com os resultados da refeição escolhida com suas devidas porções e demonstração das calorias por refeição, lembrfando que esse resultado deverá ser proporcional ao peso de cada alimento, os alimentos deverão ser consultados nos principais sites que oferecem dietas saudaveis.
